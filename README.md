# 🤹🏻‍♂️ Agent Skills

A collection of agent skills for use with GitHub Copilot and other AI agents.

## Installation

Install the `skills` CLI via Homebrew:

```sh
brew install skills
```

## Usage

Browse and install skills from this collection using the `skills` CLI:

```sh
# List globally available skills
skills list -g

# Install skill globally
skills install -g jamesrwhite/skills
```

## Skills

| Skill | Description |
|-------|-------------|
| [hello](skills/hello/) | Greets the user and demonstrates basic skill functionality |
| [goodbye](skills/goodbye/) | Says goodbye to the user and demonstrates basic skill functionality |
